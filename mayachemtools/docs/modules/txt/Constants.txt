NAME
    Constants

SYNOPSIS
    use Constants;

    use Constants qw(:all);

    use Constants qw(:math);

DESCRIPTION
    Constants module provided the following constants:

    Pi, TwoPi, PiByTwo, PiByFour, InversePi, OneByPi, SqrtTwoPi,
    OneBySqrtTwoPi, InverseSqrtTwoPi, Phi, E
